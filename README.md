# Daylight Dashboard

Daylight Dashboard is the reusable product/app for private daily companion pages.

The first loved-one page is:

- Page name: Mary's Daily Companion
- Gift domain: MyDaughterLovesMe.com
- Product domain: DaylightDashboard.com

## How To Open Locally

Open `index.html` in a browser.

For the Mary page demo, open:

```txt
index.html#mary
```

Temporary demo PIN:

```txt
mary
```

## What This Version Does

- Shows Daylight Dashboard as the product homepage.
- Opens Mary's Daily Companion directly when the domain is `MyDaughterLovesMe.com`.
- Includes a simple PIN screen for the Mary page.
- Documents the domain setup for Vercel.

## Important Privacy Note

This first version is a static prototype. The PIN is appropriate for a private-feeling gift page, but it is not strong security because the page runs fully in the browser.

Before adding sensitive medical, financial, or deeply personal family information, add real account login and server-side access control.

## Future Product Shape

- `DaylightDashboard.com`: product homepage and account login.
- `MyDaughterLovesMe.com`: Mary's private gift URL.
- Customer-owned domains: point directly to each loved one's page.
- Future account routes: family admin, daily updates, memory library, caregiver notes, and secure login.
